Part 3

ComfyUI 工作流技術說明

工作流介紹：基本生圖、局部重繪、XY軸 LoRA 測試、LoRA 自動打標和歸檔
01 基本生圖

工作流總覽

第一次生圖

兩倍放大

Image 預覽

工作流說明
Model：載入模型、LoRA 與 VAE，clip skip 也可在此調整
Prompt：合併提詞可將不常變動的提詞分離，按部位撰寫
ControlNet：常用 canny、depth、openpose；準確率 canny > depth > openpose
第一次生圖：step 25～30，cfg 5.5～7.5
兩倍放大：降躁值需降低，過高會導致和原圖相差過大
02 局部重繪

工作流總覽

Model

Gemini

Prompt

ControlNet

KSampler

Crop & Stitch

工作流說明
Gemini：整合 **Gemini 大型語言模型（LLM）** 進行遮罩處理後圖像的解析，利用 **Gemini-2.5-flash 節點** 實現自動化的 Prompt 工程
ControlNet：局部重繪使用準確度最高的 canny
KSampler：降躁值常用 0.5、0.3、0.1、0.01；cfg 可調整色彩飽和度
Crop & Stitch：遮罩裁剪 → 放大重繪 → 縮回拼貼
03 XY 軸 LoRA 測試

工作流總覽

Prompt

LoRA × Weight

LoRA × Seed

KSampler

工作流說明
Prompt：功能與基本生圖相同，合併提詞分離固定與變動的部分
LoRA × Weight：儲存 8 個訓練版本，最終採用 5～7 階段；測試各階段在 weight 0.5～1 的表現
LoRA × Seed：測試不同訓練版本間的差異
KSampler：參數預設即可，無需額外調整
04 LoRA 自動打標和歸檔

工作流總覽

自製 Gemini 節點說明
自動讀取指定資料夾內圖片，分析打標後按順序和檔名整齊排列至另一個資料夾。
使用 Gemini API Key，設計可同時載入多把金鑰，被限流時自動切換至下一把。
